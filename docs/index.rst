Solium
======

Solium is a Linter for the Solidity programming language.

Its a fully customizable static analyzer that allows you to create your own Solidity style specifications (or borrow someone else's) with 0 coding.

Enforce patterns and practices in smart contracts across your organization and catch bugs before they make it to ``master``!
